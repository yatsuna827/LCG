# LCG.py

## Description
ポケモンの乱数調整用のLCGクラスを提供するパッケージです。

## How to install
pip: `pip install git+https://github.com/yatsuna827/LCG.py@main`
poetry: `poetry add git+https://github.com/yatsuna827/LCG.py#main`

## Commands
test: `python -m unittest discover tests`
